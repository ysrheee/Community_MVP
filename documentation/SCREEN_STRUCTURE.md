# 화면 구성 (Screen Structure)

## 1. 전체 화면 구조

```
┌─────────────────────────────────────────────────────────┐
│                    랜딩페이지                            │
│                   (Landing Page)                        │
└─────────────────────┬───────────────────────────────────┘
                      │
          ┌───────────┴───────────┐
          ▼                       ▼
┌─────────────────┐     ┌─────────────────┐
│    회원가입      │     │     로그인      │
│   (Sign Up)     │     │   (Sign In)     │
└────────┬────────┘     └────────┬────────┘
         │                       │
         └───────────┬───────────┘
                     ▼
         ┌─────────────────────┐
         │    인적사항 입력     │
         │  (Profile Setup)    │
         └──────────┬──────────┘
                    ▼
         ┌─────────────────────┐
         │    구독 결제        │
         │   (Subscription)    │
         └──────────┬──────────┘
                    ▼
         ┌─────────────────────┐
         │      대시보드       │
         │    (Dashboard)      │
         └──────────┬──────────┘
                    │
     ┌──────────────┼──────────────┐
     ▼              ▼              ▼
┌─────────┐  ┌───────────┐  ┌──────────┐
│ 모임신청 │  │ 모임정보  │  │ 마이페이지│
│ (Apply) │  │ (Details) │  │ (MyPage) │
└─────────┘  └───────────┘  └──────────┘
```
