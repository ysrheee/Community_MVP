# 유저 스토리

## 1. 사용자 페르소나

### 페르소나 A: 직장인 민지 (29세)
- 서울 성수동 거주
- IT 회사 마케터
- 평일 바빠서 새로운 사람 만날 기회가 없음
- 주말엔 쉬고 싶지만 사회적 연결도 원함

### 페르소나 B: 프리랜서 준혁 (34세)
- 서울 망원동 거주
- 프리랜서 개발자
- 재택근무로 고립감 느낌
- 동네에서 가볍게 만날 사람들이 필요

### 페르소나 C: 이직한 수연 (31세)
- 서울 합정동 최근 이사
- 새 동네에 아는 사람이 없음
- 동네 친구를 사귀고 싶음

## 3. 유저 스토리 상세

## Epic 1: 서비스 발견 및 이해

### US-1.1: 서비스 내용 확인
**As a** 잠재 고객
**I want to** 랜딩페이지에서 서비스가 무엇인지 빠르게 이해하고 싶다
**So that** 나에게 맞는 서비스인지 판단할 수 있다

**인수 조건:**
- [ ] 히어로 섹션에서 핵심 가치가 3초 내 전달됨
- [ ] 서비스 진행 방식이 명확히 설명됨
- [ ] 가격 정보가 투명하게 공개됨

---

### US-1.2: 가격 및 포함 내역 확인
**As a** 잠재 고객
**I want to** 구독료에 무엇이 포함되고 별도 비용이 무엇인지 알고 싶다
**So that** 예산을 계획할 수 있다

**인수 조건:**
- [ ] 월 구독료 29,900원 명시
- [ ] 식사/음료비 별도 안내
- [ ] 구독으로 얻는 혜택 명시

---

### US-1.3: FAQ 확인
**As a** 잠재 고객
**I want to** 자주 묻는 질문과 답변을 확인하고 싶다
**So that** 궁금한 점을 해소하고 신청 여부를 결정할 수 있다

**인수 조건:**
- [ ] 최소 5개 이상의 FAQ 제공
- [ ] 아코디언 형태로 펼쳐보기 가능

---

## Epic 2: 회원가입 및 구독

### US-2.1: 회원가입
**As a** 신규 사용자
**I want to** 간편하게 회원가입하고 싶다
**So that** 빠르게 서비스를 이용할 수 있다

**인수 조건:**
- [ ] 이메일 또는 소셜 로그인 지원
- [ ] 필수 정보만 최소한으로 입력
- [ ] 개인정보 수집 동의 절차 포함

---

### US-2.2: 인적사항 입력
**As a** 신규 회원
**I want to** 나의 인적사항을 입력하고 싶다
**So that** 나와 잘 맞는 사람들과 매칭될 수 있다

**인수 조건:**
- [ ] 기본 정보: 이름, 연락처, 거주 동네
- [ ] 직업/직군 선택
- [ ] 관심사 선택 (복수 선택 가능)
- [ ] 성향 질문 (예: 외향적/내향적, 술 선호도 등)
- [ ] 자기소개 (선택)

---

### US-2.3: 구독 결제
**As a** 신규 회원
**I want to** 월 구독을 결제하고 싶다
**So that** 금요일 모임에 참가할 자격을 얻을 수 있다

**인수 조건:**
- [ ] 결제 금액 29,900원 명시
- [ ] 카드 결제 지원
- [ ] 자동 결제 동의 절차
- [ ] 결제 완료 확인 화면
- [ ] 이메일/SMS 결제 확인 발송

---

### US-2.4: 구독 해지
**As a** 구독 회원
**I want to** 구독을 해지하고 싶다
**So that** 더 이상 비용이 청구되지 않는다

**인수 조건:**
- [ ] 마이페이지에서 해지 가능
- [ ] 해지 전 확인 절차
- [ ] 해지 후에도 결제 기간 내 서비스 이용 가능
- [ ] 해지 완료 안내

---

## Epic 3: 모임 신청

### US-3.1: 이번 주 모임 신청
**As a** 구독 회원
**I want to** 이번 주 금요일 모임에 신청하고 싶다
**So that** 이번 주 저녁식사 모임에 참가할 수 있다

**인수 조건:**
- [ ] 신청 버튼 클릭으로 간편 신청
- [ ] 수요일 마감 안내 표시
- [ ] 신청 완료 확인 메시지
- [ ] 신청 내역 확인 가능

---

### US-3.2: 신청 마감 알림
**As a** 구독 회원
**I want to** 신청 마감 전 알림을 받고 싶다
**So that** 신청을 놓치지 않을 수 있다

**인수 조건:**
- [ ] 화요일 알림 발송 (마감 하루 전)
- [ ] 푸시/SMS/이메일 중 선택 가능

---

### US-3.3: 신청 취소
**As a** 모임 신청자
**I want to** 신청을 취소하고 싶다
**So that** 불참 시 다른 사람에게 기회를 줄 수 있다

**인수 조건:**
- [ ] 수요일 마감 전까지 취소 가능
- [ ] 취소 확인 절차
- [ ] 취소 완료 안내

---

## Epic 4: 모임 참여

### US-4.1: 모임 정보 확인
**As a** 모임 확정자
**I want to** 모임 장소와 시간, 함께할 멤버 정보를 확인하고 싶다
**So that** 모임에 제대로 참석할 수 있다

**인수 조건:**
- [ ] 목요일에 모임 정보 안내 발송
- [ ] 식당 이름, 주소, 지도 링크
- [ ] 모임 시간
- [ ] 함께할 멤버 간단 프로필 (이름, 직업, 한줄 소개)

---

### US-4.2: 모임 장소 찾기
**As a** 모임 참석자
**I want to** 모임 장소를 쉽게 찾아가고 싶다
**So that** 길을 헤매지 않고 도착할 수 있다

**인수 조건:**
- [ ] 지도 앱 연동 (네이버/카카오/구글맵)
- [ ] 식당 전화번호 제공

---

## Epic 5: 모임 후 활동

### US-5.1: 모임 후기 작성
**As a** 모임 참석자
**I want to** 모임 후기를 남기고 싶다
**So that** 서비스 개선에 기여하고 내 경험을 기록할 수 있다

**인수 조건:**
- [ ] 모임 다음날 후기 작성 요청 발송
- [ ] 별점 평가 (1-5점)
- [ ] 텍스트 후기 (선택)
- [ ] 익명 처리 옵션

---

### US-5.2: 재매칭 선호도 표시
**As a** 모임 참석자
**I want to** 다시 만나고 싶은/만나고 싶지 않은 사람을 표시하고 싶다
**So that** 다음 매칭에 반영될 수 있다

**인수 조건:**
- [ ] 비공개로 선호도 표시
- [ ] 매칭 알고리즘에 반영
