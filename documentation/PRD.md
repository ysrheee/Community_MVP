# PRD: 동네 금요 저녁식사 모임 서비스

## 1. 개요

### 1.1 제품 비전
동네 이웃들이 매주 금요일 저녁, 근사한 동네 맛집에서 새로운 사람들과 함께 식사하며 교류할 수 있는 커뮤니티 서비스

### 1.2 핵심 가치
- 바쁜 일상 속 의미 있는 사회적 연결 제공
- 취향이 맞는 사람들과의 자연스러운 만남
- 번거로운 예약/준비 없이 참여만 하면 되는 편리함

---

## 2. 서비스 상세

### 2.1 모임 구성
| 항목 | 내용 |
|------|------|
| 모임 일시 | 매주 금요일 저녁 |
| 모임 인원 | 6명 1그룹 |
| 모임 장소 | 주최측이 선정한 동네 맛집 |
| 매칭 방식 | 인적사항 기반 취향/성향 매칭 |

### 2.2 신청 및 결제
| 항목 | 내용 |
|------|------|
| 신청 마감 | 매주 수요일 |
| 구독 가격 | 월 29,900원 |
| 결제 방식 | 월 단위 구독 |
| 식사/음료비 | 각자 부담 |

### 2.3 역할 분담
**주최측 담당:**
- 동네 맛집 선정 및 예약
- 참가자 매칭 (인적사항 기반)
- 모임 준비 및 운영

**참가자 담당:**
- 월 구독료 결제 (29,900원)
- 본인 식사/음료비 부담
- 수요일까지 참가 신청

---

## 3. 랜딩페이지 요구사항

### 3.1 페이지 목적
- 서비스 소개 및 가치 전달
- 신규 회원 구독 신청 유도

### 3.2 필수 섹션

#### 히어로 섹션
- 서비스 핵심 메시지
- CTA 버튼 (신청하기)

#### 서비스 소개
- 모임 방식 설명
- 매칭 시스템 소개
- 주최측이 준비하는 것들

#### 가격 안내
- 월 구독료: 29,900원
- 식사/음료비 별도 안내

#### 신청 프로세스
1. 구독 신청 및 결제
2. 인적사항 입력
3. 매주 수요일까지 참가 신청
4. 금요일 모임 참석

#### 신청 폼
- 기본 정보 (이름, 연락처, 이메일)
- 인적사항 (직업, 관심사, 성향 등) - 매칭에 활용
- 거주 동네
- 결제 정보 연동

#### FAQ
- 자주 묻는 질문과 답변

#### 푸터
- 서비스 문의처
- 이용약관, 개인정보처리방침

### 3.3 디자인 방향
- 따뜻하고 친근한 분위기
- 신뢰감을 주는 깔끔한 UI
- 모바일 최적화 필수

---

## 4. 사용자 플로우

```
[랜딩페이지 방문]
       ↓
[서비스 내용 확인]
       ↓
[신청하기 버튼 클릭]
       ↓
[회원가입 / 로그인]
       ↓
[인적사항 입력]
       ↓
[구독 결제 (29,900원/월)]
       ↓
[매주 수요일까지 참가 신청]
       ↓
[금요일 모임 참석]
```

---

## 5. 향후 고려사항

- 모임 후기 및 평점 시스템
- 재매칭 선호도 반영
- 다양한 모임 테마 (취미별, 연령대별 등)
- 동네 확장

---

## 6. 성공 지표 (KPI)

- 월간 신규 구독자 수
- 구독 유지율 (Retention Rate)
- 모임 참석률
- 참가자 만족도 (NPS)
